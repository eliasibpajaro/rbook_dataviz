5.1 Matriz de correlación de Spearman

corr_df <- df |>
  select(all_of(num_cols)) |>
  drop_na()

corr_mat <- cor(corr_df, method = "spearman")

corr_long <- as.data.frame(as.table(corr_mat)) |>
  rename(var1 = Var1, var2 = Var2, corr = Freq)

ggplot(corr_long, aes(x = var1, y = var2, fill = corr)) +
  geom_tile(color = "white") +
  scale_fill_gradient2(low = "white", high = "#2166ac", mid = "white", midpoint = 0) +
  labs(
    title = "Correlación de Spearman entre variables numéricas",
    x = NULL,
    y = NULL,
    fill = "rho"
  ) +
  theme(axis.text.x = element_text(angle = 45, hjust = 1))

5.4 Respuesta por clases de suelo y grupos de magnitud

df_mag_groups <- df |>
  mutate(
    Mag_group = cut(
      Magnitude,
      breaks = c(min(Magnitude, na.rm = TRUE), 4, 5, 6, max(Magnitude, na.rm = TRUE)),
      include.lowest = TRUE,
      labels = c("low", "4–5", "5–6", "high")
    )
  )

ggplot(df_mag_groups, aes(x = Soil_Class, y = T_0.01_RotD50)) +
  geom_boxplot(
    fill = "#4F81BD",
    outlier.alpha = 0.2,
    show.legend = FALSE
  ) +
  facet_wrap(~ Mag_group, nrow = 1) +
  labs(
    title = "T_0.01_RotD50 por clase de suelo y grupos de magnitud",
    x = "Soil_Class",
    y = "T_0.01_RotD50"
  )

Este gráfico facilita ver si el efecto aparente del sitio cambia con el tamaño del evento.

5.6 Lectura integrada

En conjunto, el análisis multivariado sugiere que:

  • Magnitude y Rrup_OpenQuake concentran una porción importante de la explicación de la respuesta;
  • la profundidad hipocentral y la clase de suelo refinan la interpretación;
  • existen señales de mezcla entre subconjuntos del dataset, especialmente por la variable origen;
  • la estructura espacial confirma que la base no corresponde a un único dominio geográfico reducido.
