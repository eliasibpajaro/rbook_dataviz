Presentación

Este libro en bookdown reorganiza el material del proyecto en un formato de lectura continua.
El foco principal es el análisis exploratorio de datos (EDA) del conjunto NGACOL.csv, manteniendo la estructura conceptual del Jupyter Book original en tres bloques:

  1. Análisis univariado
  2. Análisis bivariado
  3. Análisis multivariado

0.1 Panorama del dataset

El archivo analizado contiene 10,239 registros, 403 combinaciones únicas de ubicación de evento y 4,113 ubicaciones únicas de estación.
